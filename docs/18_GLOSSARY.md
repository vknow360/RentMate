# Glossary

## Terms and Definitions

- **Cloudinary:** A cloud-based service used by RentMate to securely store, manage, and optimize property and user profile images.
- **Cold Start:** The delay experienced when an application (hosted on a free-tier PaaS like Render) handles its first request after being spun down due to inactivity.
- **Gemini 2.0 Flash:** A lightweight Google Generative AI model used by RentMate to analyze student lifestyle preferences and generate a roommate compatibility score.
- **Glassmorphism:** A UI design trend characterized by semi-transparent backgrounds with background blur (frosted glass effect). Used sparingly in RentMate to provide depth.
- **JWT (JSON Web Token):** A secure, stateless method for transmitting information between the client and backend. Used in RentMate to authenticate users and manage sessions.
- **Mongoose:** An Object Data Modeling (ODM) library for MongoDB and Node.js. It manages relationships between data, provides schema validation, and translates between objects in code and the representation of those objects in MongoDB.
- **MVP (Minimum Viable Product):** A version of the platform containing only the core features necessary to fulfill the primary business objectives, allowing for rapid deployment and user feedback.
- **PaaS (Platform as a Service):** Cloud computing services (like Render and Vercel) that provide a platform allowing customers to develop, run, and manage applications without the complexity of building and maintaining infrastructure.
- **RBAC (Role-Based Access Control):** A method of restricting network access based on the roles of individual users. In RentMate, roles include `student`, `owner`, and `admin`.
- **SPA (Single Page Application):** A web application or website that interacts with the user by dynamically rewriting the current web page with new data from the web server, instead of the default method of a web browser loading entire new pages. RentMate's frontend is an SPA.
